# 实践1.1 学习 Flex：用 Flex 实现词法分析

本实践已在当前目录下完成一个独立的 Flex 词法分析器，核心文件为：

```text
lexer.l
Makefile
tests/lex_ok.cmm
tests/lex_error.cmm
```

## 实现内容

`lexer.l` 使用 Flex 描述 C-- 语言的词法规则，可以识别以下词法单元：

- 关键字：`int`、`float`、`struct`、`return`、`if`、`else`、`while`
- 标识符：`ID`
- 类型：`TYPE: int`、`TYPE: float`
- 整数：十进制、八进制、十六进制，统一按十进制数值输出
- 浮点数：普通小数和指数形式浮点数
- 运算符和界符：`ASSIGNOP`、`RELOP`、`PLUS`、`MINUS`、`STAR`、`DIV`、`AND`、`OR`、`NOT`、`DOT`、`SEMI`、`COMMA`、括号、方括号和花括号
- 注释：过滤 `//` 行注释和 `/* ... */` 块注释

程序同时实现了词法错误 A 的报告，包括：

- 未定义字符，例如 `~`
- 非法八进制数，例如 `09`
- 非法十六进制数，例如 `0x3G`
- 非法指数浮点数，例如 `1.05e`
- 未闭合的块注释

错误输出格式遵守题目要求：

```text
Error type A at Line [行号]: [说明文字].
```

## 编译与运行

在当前目录执行：

```bash
make clean
make
```

生成可执行文件：

```text
lexer
```

运行方式：

```bash
./lexer tests/lex_ok.cmm
./lexer tests/lex_error.cmm
```

也可以分析任意 C-- 源文件：

```bash
./lexer your_source_file.cmm
```
